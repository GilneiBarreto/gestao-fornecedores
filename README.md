# Explorando Padrões de Projetos na Prática com Java

Repositório com as implementações dos padrões de projeto explorados no curso "Santander 2024 - Backend com Java". Esta é uma implementação que visa contemplar o desafio "Design Patterns com Java: Dos Clássicos (GoF) ao Spring Framework".
O projeto explora alguns padrões de projeto vistos no curso e que estão presentes no Spring Boot. São eles:
- Singleton
- Strategy/Repository
